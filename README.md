# Nearby Places Finder 🌍

## 🌐 Live Demo
https://nearby-project-alpha.vercel.app

A smart web application to discover nearby places with categorized recommendations, ratings, and a clean responsive UI.

## 🚀 Features
- 📍 Explore nearby places
- ⭐ Ratings display
- 🖼️ Image preview with fallback support
- 📱 Responsive design
- 🗂️ Categorized suggestions (food, chill, etc.)

## 🛠️ Tech Stack
- React.js
- JavaScript
- CSS

## ▶️ Run Locally
```bash
npm install
npm start
